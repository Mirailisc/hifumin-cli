# hifumin-cli

a cli tool for [Hifumin](https://hifumin.app) inspired by [Mangal](https://github.com/metafates/mangal)

## How to use it ?

### By Package execute

npm

```bash
npx hifumin-cli
```

pnpm

```bash
pnpx hifumin-cli
```

### By Package manager

Clone this repo

#### Install dependencies

```bash
pnpm install
```

#### Start cli

```bash
pnpm start
```
